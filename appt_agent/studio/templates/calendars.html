  1. Ve a console.cloud.google.com y crea un proyecto
  2. Habilita la Google Calendar API
  3. Credenciales → Crear credenciales → ID de cliente OAuth 2.0
  4. Tipo: Aplicación web
  5. En "URI de redireccionamiento autorizados" agrega exactamente:
    {{ callback_uri }}
  6. OAuth consent screen → Test users → agrega tu correo
  7. Descarga el JSON y súbelo aquí abajo

¿Cuándo usar Service Account?

Ideal para servidores. No requiere interacción del usuario. Necesitas compartir el calendario con el email de la cuenta de servicio.

  1. En Google Cloud Console → IAM → Cuentas de servicio → Crear
  2. Descarga la clave JSON
  3. En Google Calendar → Configuración del calendario → Compartir con personas específicas
  4. Agrega el email de la cuenta de servicio con permiso Realizar cambios en eventos
